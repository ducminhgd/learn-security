# Day 311 — CTF Web Competition Day 1: Warm-Up and Recon Phase

> "Competitions are honest. There is no excuse, no network issue to blame, no
> 'I would have found it with more time.' The scoreboard is the truth. That is
> why you practise them."
>
> — Ghost

---

## Goals

Complete a timed CTF session targeting web category challenges.
Focus on warm-up challenges to build momentum and calibrate difficulty.

**Time budget:** 4–5 hours (simulate a competition sprint).

---

## Competition Setup

```
Platform:  CTFtime.org / HackTheBox CTF / PicoCTF / CTF event of choice
Date:      ___
Team:      Solo
Target:    Web category

Competition ground rules:
  - No hints for the first 30 minutes on any challenge
  - Document every dead end — that is the learning
  - Prioritise challenges in order: most solves → fewer solves
  - If stuck > 45 min without progress: move on, come back
```
