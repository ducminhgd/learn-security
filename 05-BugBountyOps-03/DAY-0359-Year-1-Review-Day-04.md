## Gap Closure Protocol

For each gap, apply this 45-minute drill:

```
1. Read (10 min): One definitive source — first principles only.
   No tools. Understand the root cause.

2. Lab (25 min): One PortSwigger or HTB lab targeting this exact gap.
   Complete it without hints for the first 20 minutes.

3. Explain (10 min): Write a 3-sentence explanation as if teaching
   a junior colleague. If you can't write it, you haven't learned it.
```
