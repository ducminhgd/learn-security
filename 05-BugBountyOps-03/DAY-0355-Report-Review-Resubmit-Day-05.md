## Year-1 Gate Eligibility Pre-Check

The Bug Bounty Hunter Gate (Day 365) requires demonstrated competency.
Use today to check readiness:

```
Gate criteria (from SYLLABUS):
  [ ] Can conduct a full web application pentest end-to-end
  [ ] Can write a professional finding report (title, CVSS, impact, repro, fix)
  [ ] Can read and apply a programme policy correctly (scope, OOS, safe harbour)
  [ ] Can run and interpret automated recon (subfinder, httpx, Nuclei)
  [ ] Can chain two vulnerabilities into a higher-severity finding
  [ ] Can negotiate severity with evidence and a CVSS argument
  [ ] Understands the legal framework (CFAA, Computer Misuse Act)

Self-assessment:
  Criteria I clearly meet: ___
  Criteria I am uncertain about: ___
  Criteria I need to evidence before Day 365: ___
```

## Year-1 Review Block Agenda (Days 356–360)

```
Day 356: Foundation and offensive skill recap — can I explain every technique
         I used in the past year from first principles?

Day 357: Defensive translation — for every offensive technique I know,
         can I describe the detection rule and the fix?

Day 358: Full-year retrospective — metrics, technique radar, skill progression

Day 359: Gap closure — any remaining weak areas before the gate

Day 360: Gate simulation — dry run of the Year-1 Gate format
```
